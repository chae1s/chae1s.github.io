# 글씨/문장 강조
- 이텔릭(Italic)   
- 굵은 글씨(Bold)   
- 밑줄 긄씨(Underline)   
- 모노스페이스(Monospace)   
- 라인 블럭(Line block)   
- 체크 박스(Task lists)   
- 글자 색상 - html 태그 사용   
```
_이탤릭체 글씨_   
*이탤릭체 글씨*   

__굵은 글씨__   
**굵은 글씨**   

**_굵은 이탤릭체 글씨_**   

<u>밑줄 글씨</u>   

~~취소선 글씨~~   

`모노스페이스`   

| 첫 번째 라인 블럭   
| 두 번째 라인 블럭   

- [x] 이것은 체크 박스 입니다.
- [ ] 이것은 비어있는 체크 박스 입니다.
```
